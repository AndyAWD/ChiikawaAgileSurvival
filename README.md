## 這是什麼？

這是一個**可以在自己電腦上開起來玩的互動簡報**。你會看到：

- 一張職場冒險地圖，上面有 7 個關卡
- 每一關都有吉伊卡哇角色出場，用對話帶出敏捷工作的重點
- 可以選擇從哪一關開始，完成的關卡會蓋上章戳
- 播放背景音樂（可開關）

適合：想用輕鬆方式認識敏捷（Agile）觀念的同事、朋友，或單純想看吉伊卡哇的你。

## 快速開始：觀看簡報

### Windows 使用者

1. 打開專案資料夾，找到 `start.bat`
2. 用滑鼠**左鍵雙擊** `start.bat`
3. 會跳出一個黑色的命令視窗，裡面顯示「在 http://localhost:8000 啟動靜態伺服器」，**請勿關閉這個視窗**
4. 瀏覽器會自動開啟簡報頁面
5. 如果瀏覽器沒自動開啟，請自己打開瀏覽器，在網址列輸入：`http://localhost:8000`

### macOS 使用者

1. 打開專案資料夾，找到 `start.command`
2. 用滑鼠**雙擊** `start.command`
3. **首次執行**若出現「無法開啟，因為它來自未識別的開發者」警告：
   - 關閉警告視窗
   - 用滑鼠**右鍵點選** `start.command` → 選「打開」
   - 警告視窗會多出一個「打開」按鈕，按下去即可
   - 之後就能直接雙擊使用
4. 終端機視窗會打開，**請勿關閉**
5. 瀏覽器會自動開啟簡報頁面（`http://localhost:8000`）

### 結束簡報

- 關閉瀏覽器分頁
- **一定要記得關掉命令視窗／終端機視窗**（或在視窗內按 `Ctrl + C`），否則伺服器會一直跑在背景

## 修改簡報內容：使用視覺化編輯器

如果你想改簡報裡的文字、換角色圖片、新增關卡，**不需要會寫程式**，本專案附了一個編輯器介面。

### 啟動編輯器

- **Windows**：雙擊 `start-editor.bat`
- **macOS**：雙擊 `start-editor.command`（首次執行若遇到警告，參考上面觀看簡報的步驟）

瀏覽器會自動開啟編輯頁面：`http://localhost:8001/editor.html`

### 可以修改哪些東西？

進入編輯器後，左側選單會看到以下項目：

| 選項 | 可修改內容 |
| --- | --- |
| 📋 基本資訊 | 簡報標題、副標題、標語 |
| 🔤 UI 文字 | 按鈕上顯示的文字（例如「下一關」「回地圖」） |
| 🎵 音樂設定 | 上傳或更換背景音樂（BGM）、調整音量、是否循環播放 |
| 🖼️ 背景圖片 | 標題頁、地圖頁的底圖 |
| 👥 角色設定 | 新增、刪除或修改角色（名字、所屬部門、特色、代表色、圖片） |
| 📖 關卡設定 | 每一關的標題、出場角色、對話、重點、在地圖上的位置 |

### 儲存修改

1. 編輯完畢後，點選頁面上的 **💾 儲存** 按鈕
2. 看到綠色的「已儲存」字樣就代表成功
3. 回到簡報頁面，**按 `F5` 或 `Cmd + R` 重新整理**，就能看到最新內容

### 小提醒

- 編輯器（8001 埠）與簡報播放器（8000 埠）可以同時開，但建議一次只開一個比較不會搞混
- 編輯器會直接修改專案裡的 `content.json`，改壞了可以用 Git 還原
- 上傳圖片大小限制為 20 MB

## 專案檔案結構（簡略）

```
web/
├── start.bat / start.command              ← 啟動「簡報播放」
├── start-editor.bat / start-editor.command ← 啟動「編輯器」
├── content.json                           ← 所有簡報內容（建議透過編輯器修改）
├── assets/
│   ├── images/                            ← 圖片素材（可透過編輯器上傳替換）
│   ├── css/                               ← 樣式檔（請勿修改）
│   └── js/                                ← 程式碼（請勿修改）
├── index.html                             ← 簡報主頁面（請勿修改）
├── editor.html                            ← 編輯器頁面（請勿修改）
└── editor_server.py                       ← 編輯器後端程式（請勿修改）
```

**給非工程師的提醒**：除了 `content.json` 和 `assets/images/` 裡的圖片之外，其他檔案請盡量不要直接修改，以免簡報壞掉。

## 常見問題（FAQ）

**Q：可以直接雙擊 `index.html` 開啟嗎？**
A：不行。瀏覽器為了安全考量，不允許本機網頁直接讀取本機 JSON 檔，所以一定要透過 `start.bat` 或 `start.command` 啟動本機伺服器。

**Q：我在地圖上完成的關卡章戳，要怎麼重置？**
A：在地圖頁面下方有「重置進度」按鈕，點下去會跳出確認視窗，按確定即可全部重來。

**Q：改了 `content.json` 卻沒反應？**
A：回到瀏覽器，按 `F5`（Windows）或 `Cmd + R`（macOS）重新整理頁面。

**Q：雙擊 `start.bat` 視窗閃一下就關了？**
A：代表你的 Windows 還沒安裝 Python。請到 [python.org](https://www.python.org/downloads/) 下載安裝，**安裝時務必勾選「Add Python to PATH」**，安裝完再試一次。

**Q：macOS 顯示「無法驗證開發者」怎麼辦？**
A：用**右鍵點選** `.command` 檔 → 選「打開」→ 再按一次「打開」。之後就能正常雙擊執行。

**Q：可以把這份簡報拿去公司、學校、研習活動上播放嗎？**
A：由於簡報使用了吉伊卡哇的版權角色圖像，**請勿用於任何商業、營利或正式對外活動**。個人學習、私下分享給朋友是可以的。

**Q：音樂檔可以放哪裡？**
A：透過編輯器的「音樂設定」直接上傳即可，檔案會自動放到 `assets/audio/`。支援 MP3、WAV、OGG、M4A。
